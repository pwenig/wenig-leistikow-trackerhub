## Get Things Running
1. Clone and fork
1. `bundle install`
1. `rake db:migrate`
1. `rake spec` Uses dotenv to store tracker api... tests are frozen in place with

1. create `.env` file with `TRACKER_TOKEN='123_YOURKEYHERE_789`
1. `rails s` and enjoy
